About This Tool

This DNS tester checks your connection to various public DNS servers around the world. It helps diagnose DNS issues by showing which servers are reachable from your location.

How It Works

  1. Click "Run Tests" to check against popular DNS providers
  2. View response times and success status for each server
  3. Use "Custom Test" to check specific DNS servers

Interpreting Results

  • Success means the DNS server responded
  • Failed means the server didn't respond
  • Response times under 100ms are excellent
  • Times over 300ms may indicate problems
